# 测试前前置检查规则

## 服务器状态检查

进行任何测试前，**必须严格确认**以下两台服务器均已成功启动并正常运行：

### 1. 前端开发服务器
- **端口**：5173
- **验证方式**：访问 http://localhost:5173
- **预期**：返回正常的网页内容

### 2. 后端API服务器
- **端口**：8000
- **验证方式**：访问 http://localhost:8000
- **预期**：返回正常的API响应

## 验证方法

### Windows环境验证命令
```powershell
# 检查前端服务器（5173）
Test-NetConnection -ComputerName localhost -Port 5173 -InformationLevel Quiet

# 检查后端服务器（8000）
Test-NetConnection -ComputerName localhost -Port 8000 -InformationLevel Quiet
```

### 替代验证方式
- 使用浏览器访问对应端口
- 使用 telnet 工具测试端口连通性
- 使用 curl 命令：`curl http://localhost:端口号`

## 执行条件

**重要**：只有在两台服务器**均处于正常运行状态**后，方可开始执行测试流程。若任一服务器未运行，应先启动相应服务器，然后再进行测试。
