# 3. 后端模块

## 3.1 Admin 模块（后台管理）

路径: `backend/app/admin/`

| 子目录           | 职责   | 说明                   |
| ------------- | ---- | -------------------- |
| `controller/` | 控制器层 | 处理 HTTP 请求，参数验证，业务编排 |
| `middleware/` | 中间件  | 认证检查、权限验证、操作日志记录     |
| `service/`    | 服务层  | 业务逻辑封装（AdminService） |
| `validate/`   | 验证器  | 请求参数校验规则             |
| `event/`      | 事件监听 | 操作日志事件处理             |

### 控制器清单

| 控制器                    | 路径                                      | 职责                        |
| ---------------------- | --------------------------------------- | ------------------------- |
| AuthController         | `controller/AuthController.php`         | 登录/登出/Token刷新/个人信息/修改密码   |
| UserController         | `controller/UserController.php`         | 用户 CRUD、角色分配、密码重置、导入导出    |
| RoleController         | `controller/RoleController.php`         | 角色 CRUD、菜单/按钮/接口权限分配、数据权限 |
| MenuController         | `controller/MenuController.php`         | 菜单 CRUD、按钮管理              |
| DepartmentController   | `controller/DepartmentController.php`   | 部门 CRUD、排序、状态切换           |
| ApiController          | `controller/ApiController.php`          | 接口 CRUD、分组查询              |
| DictController         | `controller/DictController.php`         | 字典类型/数据 CRUD、按编码查询        |
| DashboardController    | `controller/DashboardController.php`    | 仪表盘统计数据                   |
| LoginLogController     | `controller/LoginLogController.php`     | 登录日志查询/统计/清理              |
| OperationLogController | `controller/OperationLogController.php` | 操作日志查询/统计/清理              |
| ProfileController      | `controller/ProfileController.php`      | 个人信息/头像/密码                |

### 中间件链（按执行顺序）

```
请求 → AllowCrossDomain → AppInit → AuthCheck → RecordOperate → ApiPermission → Controller
```

| 中间件              | 职责                           |
| ---------------- | ---------------------------- |
| AllowCrossDomain | 跨域请求处理                       |
| AppInit          | 应用初始化                        |
| AuthCheck        | JWT Token 解析与用户身份验证          |
| RecordOperate    | 操作日志自动记录（POST/PUT/DELETE 请求） |
| ApiPermission    | 接口级别权限校验                     |

## 3.2 Miniapp 模块（微信小程序）

路径: `backend/app/miniapp/`

| 子目录           | 职责                     |
| ------------- | ---------------------- |
| `controller/` | 小程序端控制器（登录、首页、业务、个人中心） |
| `middleware/` | 小程序端认证中间件（MiniappAuth） |
| `service/`    | 微信服务、Token 黑名单、业务逻辑    |
| `validate/`   | 小程序端参数验证               |

## 3.3 Common 模块（公共基础）

路径: `backend/app/common/`

| 文件                 | 职责                                  |
| ------------------ | ----------------------------------- |
| BaseController.php | 控制器基类，提供 `success()`/`error()` 统一响应 |
| BaseModel.php      | 模型基类，集成软删除、自动时间戳                    |
| AdminAuth.php      | 权限认证核心类，管理用户权限/菜单/数据范围              |
| JwtToken.php       | JWT Token 生成/解析/验证/刷新               |
| SimpleCache.php    | 文件缓存实现，支持原子递增和互斥写入                  |
| BaseValidate.php   | 验证器基类                               |

## 3.4 Model 模块（数据模型）

路径: `backend/app/model/`

| 模型                  | 表名                    | 说明                  |
| ------------------- | --------------------- | ------------------- |
| User                | sys\_user             | 用户模型，隐藏 password 字段 |
| Role                | sys\_role             | 角色模型，含权限分配方法        |
| Menu                | sys\_menu             | 菜单模型，含树形构建方法        |
| MenuButton          | sys\_menu\_button     | 菜单按钮模型              |
| Department          | sys\_department       | 部门模型，含树形构建方法        |
| Api                 | sys\_api              | 接口模型，含路径匹配方法        |
| DictType            | sys\_dict\_type       | 字典类型模型              |
| DictData            | sys\_dict\_data       | 字典数据模型              |
| LoginLog            | sys\_login\_log       | 登录日志模型              |
| OperationLog        | sys\_operation\_log   | 操作日志模型              |
| WxConfig            | wx\_config            | 微信配置模型              |
| WxUser              | wx\_user              | 微信用户模型              |
| Business            | business              | 业务模型                |
| BusinessInteraction | business\_interaction | 业务交互模型              |
