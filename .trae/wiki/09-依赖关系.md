## 9.3 模块间调用关系

```
前端                          后端
─────────                    ─────────
pages/login                  AuthController
  └─ api/auth                  ├─ JwtToken (生成Token)
     └─ utils/request          ├─ LoginValidate
        └─ stores/user         ├─ SimpleCache (锁定检查)
                                └─ LoginLog (记录日志)

pages/system/user            UserController
  └─ api/user                  ├─ UserValidate
     └─ composables/           ├─ AdminAuth (数据权限)
        usePageTable           ├─ Role (角色查询)
        useTreeData            └─ Department (部门查询)

pages/system/role            RoleController
  └─ api/role                  ├─ RoleValidate
     └─ components/            ├─ Role (权限分配)
        RolePermissionModal    ├─ AdminAuth (缓存清除)
        RoleDataScopeModal     └─ Menu/Button/Api Model

router/dynamic               MenuController
  └─ transformMenusToRoutes    └─ Menu (树形构建)

middleware (后端)             请求处理链
  AuthCheck                    ├─ JwtToken (解析验证)
  RecordOperate                ├─ Event (操作日志事件)
  ApiPermission                └─ AdminAuth + Api (权限校验)
```
