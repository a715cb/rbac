# 10. 环境配置与部署

## 10.1 运行环境要求

| 组件      | 最低版本      | 推荐版本  |
| ------- | --------- | ----- |
| PHP     | 8.0+      | 8.1+  |
| MySQL   | 5.7+      | 8.0+  |
| Node.js | 16+       | 18+   |
| npm     | 8+        | 9+    |
| Redis   | 6.0+ (可选) | 7.0+  |
| Nginx   | 1.18+     | 1.24+ |

## 10.2 后端配置

### 环境变量 (`.env`)

复制 `.env.example` 为 `.env` 并修改：

```bash
# 应用
APP_DEBUG=false

# 数据库
DB_CONNECTION=mysql
DB_HOSTNAME=127.0.0.1
DB_PORT=3306
DB_DATABASE=rbac_system
DB_USERNAME=root
DB_PASSWORD=your_secure_password
DB_CHARSET=utf8mb4

# JWT（必须修改！）
JWT_SECRET=your_64_character_strong_random_secret_key_here
JWT_TTL=1440              # Access Token 有效期（分钟）
JWT_REFRESH_TTL=10080     # Refresh Token 有效期（分钟）

# Redis（可选）
REDIS_HOST=127.0.0.1
REDIS_PORT=6379
REDIS_PASSWORD=
REDIS_DB=0

# 缓存
CACHE_DRIVER=file          # file 或 redis

# 日志
LOG_LEVEL=error

# 微信小程序（可选）
WX_MINIAPP_APPID=
WX_MINIAPP_SECRET=

# 知识图谱记忆文件（可选）
MEMORY_FILE_PATH=runtime/data/knowledge_graph_memory.json
```

### JWT 密钥生成

```bash
# 方法1: 使用项目自带脚本
php setup_jwt_secret.php

# 方法2: 手动生成
php -r "echo bin2hex(random_bytes(32));"

# 方法3: OpenSSL
openssl rand -base64 32
```

## 10.3 前端配置

### 环境变量

创建 `.env.local`（参考 `.env.local.example`）：

```bash
# 应用标题
VITE_APP_TITLE=RBAC 权限管理系统

# API 代理前缀
VITE_APP_BASE_API=/api

# 后端 API 地址
VITE_APP_API_BASE_URL=http://localhost:8000

# 开发服务器端口
VITE_APP_PORT=5173

# 存储键前缀
VITE_APP_STORAGE_PREFIX=rbac_admin

# 视图组件路径前缀
VITE_APP_VIEW_COMPONENT_PREFIX=@/pages/

# 错误页面组件路径
VITE_APP_ERROR_COMPONENT_PATH=@/pages/error/404.vue
```

生产环境 `.env.production`：

```bash
VITE_APP_TITLE=RBAC System
VITE_APP_BASE_API=/api
VITE_APP_API_BASE_URL=https://api.example.com
VITE_APP_VIEW_COMPONENT_PREFIX=@/pages/
VITE_APP_ERROR_COMPONENT_PATH=@/pages/error/404.vue
```

## 10.4 启动步骤

### 后端启动

```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
composer install

# 3. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，配置数据库和 JWT 密钥

# 4. 初始化数据库
mysql -u root -p < database/migrations/001_init_schema.sql
mysql -u root -p < database/migrations/002_init_data.sql
# (可选) 微信小程序表
mysql -u root -p < database/migrations/004_wx_tables.sql

# 5. 生成 JWT 密钥
php setup_jwt_secret.php

# 6. 启动开发服务器
php think run
# 默认监听 http://localhost:8000
```

### 前端启动

```bash
# 1. 进入前端目录
cd frontend

# 2. 安装依赖
npm install

# 3. 配置环境变量
cp .env.local.example .env.local
# 编辑 .env.local，配置 API 地址

# 4. 启动开发服务器
npm run dev
# 默认监听 http://localhost:5173
```

## 10.5 生产部署

### 后端部署

```bash
# 1. 设置 APP_DEBUG=false
# 2. 设置强 JWT_SECRET
# 3. 配置 CACHE_DRIVER=redis（推荐）
# 4. 配置 LOG_LEVEL=error
# 5. 使用 PHP-FPM + Nginx 部署
```

**Nginx 配置示例**:

```nginx
server {
    listen 80;
    server_name your-domain.com;
    root /path/to/backend/public;
    index index.php;

    # API 请求转发到 PHP-FPM
    location /api/ {
        if (!-e $request_filename) {
            rewrite ^(.*)$ /index.php?s=$1 last;
        }
        fastcgi_pass unix:/run/php/php8.1-fpm.sock;
        fastcgi_index index.php;
        fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
        include fastcgi_params;
    }

    # 前端静态资源
    location / {
        root /path/to/frontend/dist;
        try_files $uri $uri/ /index.html;
    }
}
```

### 前端构建

```bash
# 构建生产版本
npm run build

# 产物在 dist/ 目录
# 部署到 Nginx 静态资源目录
```
