# 6. 关键类与核心函数

## 6.1 后端核心类

### 6.1.1 AdminAuth — 权限认证核心

**文件**: `backend/app/common/AdminAuth.php`

单例模式，管理当前用户的权限上下文。

| 方法                                            | 参数                        | 返回值     | 说明                      |
| --------------------------------------------- | ------------------------- | ------- | ----------------------- |
| `instance()`                                  | -                         | `self`  | 获取单例实例                  |
| `setUser(int $userId)`                        | 用户ID                      | `void`  | 设置当前用户并加载角色             |
| `getUserId()`                                 | -                         | `int`   | 获取当前用户ID                |
| `getUser()`                                   | -                         | `?User` | 获取当前用户模型                |
| `getRoles()`                                  | -                         | `array` | 获取当前用户角色列表              |
| `check(string\|array $permission, int $type)` | 权限标识, 类型(1=自定义/2=菜单/3=接口) | `bool`  | 检查权限                    |
| `isSuperAdmin()`                              | -                         | `bool`  | 判断是否超级管理员               |
| `getMenuCodes()`                              | -                         | `array` | 获取用户菜单权限码列表             |
| `getApiCodes()`                               | -                         | `array` | 获取用户接口权限码列表             |
| `getMenuTree()`                               | -                         | `array` | 获取用户菜单树                 |
| `getDataScope()`                              | -                         | `int`   | 获取数据权限范围                |
| `getDataScopeDeptIds()`                       | -                         | `array` | 获取数据权限范围部门ID列表（自定义数据权限） |
| `getScopedDeptIds()`                          | -                         | `array` | 获取数据权限范围内的部门ID列表        |
| `isSelfOnly()`                                | -                         | `bool`  | 是否仅本人数据权限               |
| `clearCache()`                                | -                         | `void`  | 清除当前用户权限缓存              |
| `clearAllCache()`                             | -                         | `void`  | 清除所有用户权限缓存              |

**使用示例**:

```php
$auth = AdminAuth::instance();
$auth->setUser($userId);

if ($auth->isSuperAdmin()) {
    // 超级管理员逻辑
}

if ($auth->check('system_user:add', 2)) {
    // 有用户新增菜单权限
}

$scopedDeptIds = $auth->getScopedDeptIds();
// 用于数据查询过滤
```

### 6.1.2 JwtToken — JWT 令牌管理

**文件**: `backend/app/common/JwtToken.php`

| 方法                         | 参数            | 返回值      | 说明            |
| -------------------------- | ------------- | -------- | ------------- |
| `generate(array $payload)` | 载荷数据          | `string` | 生成 JWT Token  |
| `parse(string $token)`     | Token 字符串     | `array`  | 解析 Token，返回载荷 |
| `validate(string $token)`  | Token 字符串     | `bool`   | 验证 Token 有效性  |
| `refresh(string $token)`   | Refresh Token | `string` | 刷新 Token      |
| `clearCache()`             | -             | `void`   | 清除 JWT 缓存     |

**Token 载荷结构**:

```json
{
  "user_id": 1,
  "username": "admin",
  "realname": "管理员",
  "roles": ["super_admin"],
  "iat": 1715000000,
  "nbf": 1715000000,
  "exp": 1715086400
}
```

**Refresh Token 载荷**额外包含 `"type": "refresh"`。

### 6.2.2 request — HTTP 请求封装

**文件**: `frontend/src/utils/request.ts`

基于 Axios 封装，核心特性：

- **自动 Token 注入**: 请求拦截器自动添加 `Authorization: Bearer {token}`
- **Token 自动刷新**: 401 响应时自动使用 Refresh Token 刷新，刷新期间队列等待
- **请求取消**: 路由切换时自动取消未完成请求
- **统一错误处理**: 按状态码映射中文错误消息

| 导出函数                            | 参数            | 返回值                       | 说明             |
| ------------------------------- | ------------- | ------------------------- | -------------- |
| `get<T>(url, params, config)`   | URL, 查询参数, 配置 | `Promise<ApiResponse<T>>` | GET 请求         |
| `post<T>(url, data, config)`    | URL, 请求体, 配置  | `Promise<ApiResponse<T>>` | POST 请求        |
| `put<T>(url, data, config)`     | URL, 请求体, 配置  | `Promise<ApiResponse<T>>` | PUT 请求         |
| `del<T>(url, config)`           | URL, 配置       | `Promise<ApiResponse<T>>` | DELETE 请求      |
| `cancelPendingRequests(reason)` | 原因            | `void`                    | 取消所有待处理请求      |
| `cancelRequest(url)`            | 请求URL         | `void`                    | 取消指定URL的请求     |
| `default (request)`             | -             | `AxiosInstance`           | Axios 实例（默认导出） |
