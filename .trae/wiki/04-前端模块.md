# 4. 前端模块

路径: `frontend/src/`

| 目录             | 职责      | 说明                                                             |
| -------------- | ------- | -------------------------------------------------------------- |
| `api/`         | API 接口层 | 按模块封装后端 API 调用                                                 |
| `assets/`      | 静态资源    | 图标、Logo 等 SVG 资源                                               |
| `components/`  | 公共组件    | Breadcrumb、Button、Captcha、Icon、TableSetting、TokenProvider      |
| `composables/` | 组合式函数   | usePageTable、useMenuTree、useDeptTree、useTreeSearch、useSortable |
| `config/`      | 配置      | 应用配置（AppConfig）、路由配置                                           |
| `constants/`   | 常量      | 存储键名等常量定义                                                      |
| `directives/`  | 自定义指令   | v-auth 权限指令、v-auth-disabled 禁用指令                               |
| `layouts/`     | 布局组件    | DefaultLayout、Sidebar、Header、TagsView、SettingDrawer            |
| `pages/`       | 页面组件    | 按功能模块划分的业务页面                                                   |
| `router/`      | 路由      | 静态路由 + 动态路由（基于后端菜单数据）                                          |
| `stores/`      | 状态管理    | Pinia Store（用户状态、权限）                                           |
| `styles/`      | 样式      | 全局样式、布局样式、CSS 变量                                               |
| `types/`       | 类型定义    | TypeScript 类型声明                                                |
| `utils/`       | 工具函数    | request（Axios封装）、token、storage、common、dom                      |

## 页面模块

| 页面路径                       | 功能          |
| -------------------------- | ----------- |
| `pages/login/`             | 登录页         |
| `pages/dashboard/`         | 仪表盘         |
| `pages/system/user/`       | 用户管理        |
| `pages/system/role/`       | 角色管理        |
| `pages/system/dept/`       | 部门管理        |
| `pages/system/dict/`       | 字典管理        |
| `pages/permission/menu/`   | 菜单管理        |
| `pages/permission/api/`    | 接口管理        |
| `pages/permission/button/` | 按钮管理        |
| `pages/monitor/login/`     | 登录日志        |
| `pages/monitor/operation/` | 操作日志        |
| `pages/error/`             | 403/404 错误页 |
