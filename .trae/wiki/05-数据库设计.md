## 5.2 核心表结构

| 表名                     | 说明      | 关键字段                                                    |
| ---------------------- | ------- | ------------------------------------------------------- |
| `sys_user`             | 用户表     | id, username, password, dept\_id, status                |
| `sys_role`             | 角色表     | id, name, code, data\_scope, data\_scope\_dept\_ids     |
| `sys_user_role`        | 用户-角色关联 | user\_id, role\_id                                      |
| `sys_menu`             | 菜单表     | id, parent\_id, name, code, path, component, menu\_type |
| `sys_menu_button`      | 菜单按钮表   | id, menu\_id, name, code                                |
| `sys_role_menu`        | 角色-菜单关联 | role\_id, menu\_id                                      |
| `sys_role_menu_button` | 角色-按钮关联 | role\_id, menu\_button\_id                              |
| `sys_role_api`         | 角色-接口关联 | role\_id, api\_id                                       |
| `sys_api`              | 接口表     | id, menu\_id, name, code, method, path                  |
| `sys_department`       | 部门表     | id, parent\_id, name, code, leader                      |
| `sys_dict_type`        | 字典类型表   | id, name, code, type                                    |
| `sys_dict_data`        | 字典数据表   | id, dict\_type\_id, label, value                        |
| `sys_login_log`        | 登录日志    | id, username, ip, status, login\_time                   |
| `sys_operation_log`    | 操作日志    | id, user\_id, module, action, method, url               |
| `sys_config`           | 系统配置    | id, name, code, value                                   |

**菜单类型 (`menu_type`)**: 1=目录, 2=菜单, 3=按钮

**数据权限 (`data_scope`)**: 1=全部数据, 2=本部门, 3=本部门及以下, 4=仅本人, 5=自定义
