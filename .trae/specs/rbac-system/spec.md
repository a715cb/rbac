# RBAC 权限系统 - 系统架构大纲

## 1. 项目概述

### 1.1 项目定位
通用可复用企业级 RBAC 权限底座系统，统一支撑自动化办公、企业网站、微信公众号、微信小程序等多项目权限鉴权。

### 1.2 核心特征
- **高通用性**：抽象通用权限模型，适配多种业务场景
- **强可移植性**：模块化设计，支持快速集成到新项目
- **标准化**：遵循行业最佳实践，统一接口规范
- **低耦合**：前后端分离，分层架构，模块间解耦

### 1.3 技术栈
#### 前端
- Vue 3.4+ (Composition API)
- TypeScript 5.0+
- Vite 5.0+
- Ant Design Vue 4.0+
- Pinia 2.0+

#### 后端
- ThinkPHP 8.0+
- MySQL 8.0+
- Redis
- JWT 认证

## 2. 系统架构

### 2.1 整体架构
```
多项目接入层 → 统一权限网关 → 业务服务层 → 数据访问层
```

### 2.2 分层架构
- **前端分层**: API 层、组件层、路由层、状态管理层
- **后端分层**: 控制器层、服务层、模型层、中间件层

## 3. 核心模块

### 3.1 认证模块
- JWT Token 认证
- Token 无感刷新
- 登录会话管理

### 3.2 用户管理
- 用户 CRUD
- 用户状态管理
- 角色分配

### 3.3 角色管理
- 角色 CRUD
- 权限分配（菜单/按钮/接口）
- 数据权限配置

### 3.4 菜单管理
- 菜单树形结构
- 动态路由生成
- 按钮权限控制

### 3.5 日志管理
- 登录日志
- 操作日志

## 4. 数据库设计

### 4.1 核心数据表（14 张）
1. sys_user - 用户表
2. sys_department - 部门表
3. sys_role - 角色表
4. sys_user_role - 用户角色关联表
5. sys_menu - 菜单表
6. sys_role_menu - 角色菜单关联表
7. sys_menu_button - 菜单按钮表
8. sys_role_menu_button - 角色菜单按钮关联表
9. sys_api - 接口表
10. sys_role_api - 角色接口关联表
11. sys_login_log - 登录日志表
12. sys_operation_log - 操作日志表
13. sys_dict_type - 字典类型表
14. sys_dict_data - 字典数据表
