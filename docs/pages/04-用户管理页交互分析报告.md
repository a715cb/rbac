# RBAC 权限管理系统 - 用户管理页交互分析报告

**文档版本**：1.0 | **生成日期**：2026-05-12 | **页面路径**：`src/pages/system/user/index.vue`

## 一、页面概述

### 1.1 页面定位

用户管理页是系统核心管理页面，采用左右分栏布局：左侧为部门树筛选器，右侧为搜索表单+数据表格。支持用户的全生命周期管理。

### 1.2 页面功能模块划分

| 模块 | 位置 | 功能 |
|------|------|------|
| 部门树筛选器 | 左侧 5/24 | 按部门层级筛选用户列表 |
| 搜索表单区 | 右侧顶部 | 关键词/状态/性别多维度筛选 |
| 表格工具栏 | 表格上方 | 新增、导出按钮 + 列设置 |
| 数据表格 | 右侧主体 | 分页用户列表展示 |
| 用户表单弹窗 | 模态层 | 新增/编辑用户信息 |
| 重置密码弹窗 | 模态层 | 重置指定用户密码 |

### 1.3 依赖组件

| 组件 | 用途 |
|------|------|
| `DeptTree` | 部门树选择器 |
| `UserFormModal` | 用户新增/编辑表单弹窗 |
| `ResetPasswordModal` | 密码重置弹窗 |
| `TableSetting` | 表格列显隐/密度/全屏设置 |
| `DictSelect` | 字典下拉选择器（状态、性别） |
| `DictTag` | 字典标签渲染 |
| `usePageTable` | 表格设置状态管理 |
| `useDict` | 字典数据缓存 |

## 二、用户操作流程

### 2.1 页面初始化流程

```
onMounted → fetchData()
    │
    ├── loading = true
    ├── 调用 getUserList({ page:1, limit:10, ...searchForm })
    │   ├── 成功 → tableData 更新 → pagination.total 更新
    │   └── 失败 → console.error (DEV) → 请求拦截器显示错误
    └── loading = false
```

### 2.2 搜索筛选流程

```
用户修改筛选条件 (关键词/状态/性别/部门)
    │
    ├── 直接修改 searchForm 或 selectedDeptId
    │
    ├── hasPendingChanges 计算属性自动检测变更
    │   └── true → 查询按钮显示脉冲动画 (CSS @keyframes pulse-pending)
    │
    ├── 用户点击"查询"按钮 → handleSearch()
    │   ├── 同步 appliedFilters 快照
    │   ├── pagination.current = 1
    │   └── fetchData(true) → 静默模式刷新
    │
    └── 用户点击"重置"按钮 → handleReset()
        ├── 清空所有 searchForm 字段
        ├── selectedDeptId = undefined
        ├── deptTreeRef.resetSelection()
        ├── 重置 appliedFilters 快照
        └── fetchData(true) → 静默模式刷新
```

### 2.3 部门树筛选流程

```
用户点击部门树节点
    │
    ▼
handleDeptSelect(deptId)
    │
    ├── selectedDeptId = deptId
    ├── pagination.current = 1
    └── fetchData(true) → 静默模式刷新（无loading动画）
```

### 2.4 新增用户流程

```
用户点击"新增"按钮
    │
    ▼
handleAdd()
    ├── currentRecord = null (新增模式标识)
    └── modalVisible = true
        │
        ▼
UserFormModal 打开
    ├── watch visible → fetchRoles() + fetchDepts() (并行加载)
    ├── nextTick → loadFormData() (加载默认值)
    ├── 用户填写表单
    │   ├── 用户名校验 (字母开头/3-20字符)
    │   ├── 密码校验 (6-20位) / 确认密码一致性
    │   ├── 部门多选 (TreeSelect + Tag展示 + 主部门设置)
    │   └── 角色多选 (Select multiple)
    ├── 用户点击确定
    │   ├── formRef.validate() → 失败返回
    │   ├── loading = true
    │   ├── createUser(formState) → 成功 → emit('success')
    │   └── loading = false
    └── emit('update:visible', false) → 弹窗关闭
```

### 2.5 编辑用户流程

```
用户点击表格行"编辑"按钮
    │
    ▼
handleEdit(record)
    ├── currentRecord = record (编辑模式标识)
    └── modalVisible = true
        │
        ▼
UserFormModal 打开 (编辑模式)
    ├── 用户名输入框 disabled (不可修改)
    ├── 密码/确认密码字段隐藏
    ├── 表单预填充 record 数据
    └── 提交时调用 updateUser(id, data)
```

### 2.6 删除用户流程

```
用户点击"删除"按钮
    │
    ▼
Popconfirm 二次确认弹窗
    ├── 取消 → 无操作
    └── 确定 → handleDelete(record)
        ├── deleteUser(record.id)
        ├── 成功 → message.success + fetchData()
        └── 失败 → console.error(DEV) + 请求拦截器显示错误
```

### 2.7 状态切换流程

```
用户点击行内 Switch 开关
    │
    ▼
handleStatusChange(record, checked)
    ├── changeUserStatus(record.id, checked ? 1 : 0)
    ├── 成功 → message.success + fetchData() (刷新列表)
    └── 失败 → message.error + fetchData() (回滚显示)
```

### 2.8 CSV导出流程

```
用户点击"导出"按钮
    │
    ▼
handleExport()
    ├── exportUsers(当前筛选条件) → 获取全量数据
    ├── 构建表头行 + 数据行
    ├── escapeCsvField() 转义特殊字符
    ├── 字典转换 (性别/状态 → 可读文本)
    ├── Blob创建 + BOM头添加 (\ufeff)
    ├── 创建临时 <a> 标签触发下载
    └── URL.revokeObjectURL() 释放内存
```

## 三、交互触发条件与状态反馈

### 3.1 筛选条件变更提示

| 触发条件 | 交互反馈 |
|----------|----------|
| 修改关键词 | hasPendingChanges=true → 查询按钮脉冲动画 |
| 修改状态下拉 | hasPendingChanges=true → 查询按钮脉冲动画 |
| 修改性别下拉 | hasPendingChanges=true → 查询按钮脉冲动画 |
| 点击部门树节点 | hasPendingChanges=true → 查询按钮脉冲动画 |
| 点击"查询"按钮 | 同步快照 → hasPendingChanges=false → 动画消失 |

### 3.2 表格交互状态

| 操作 | 加载状态 | 刷新模式 |
|------|----------|----------|
| 初始加载 | loading=true → 表格 loading | 正常模式 |
| 部门树选择 | 无loading动画 | 静默模式(silent=true) |
| 搜索/重置 | 无loading动画 | 静默模式(silent=true) |
| 分页切换 | loading=true | 正常模式 |
| 增删改操作 | loading=true | 正常模式 |

### 3.3 表格列自定义渲染

| 列名 | 渲染方式 | 交互特性 |
|------|----------|----------|
| 头像 | Avatar组件 + UserOutlined兜底 | 纯展示 |
| 性别 | getDictLabel + Tag颜色映射 | 蓝色=男，粉色=女 |
| 部门 | Tag列表，主部门蓝色+标注"(主)" | 纯展示 |
| 角色 | 蓝色Tag列表，Space自动换行 | 纯展示 |
| 状态 | Switch开关 | 点击实时切换状态 |
| 操作 | 编辑/重置密码/删除按钮 | 删除需Popconfirm确认 |

### 3.4 分页交互

| 触发条件 | 行为 |
|----------|------|
| 切换页码 | pagination.current 更新 → fetchData() |
| 切换每页条数 | pagination.pageSize 更新 → fetchData() |
| 搜索/筛选 | pagination.current 重置为 1 → fetchData(true) |

## 六、异常处理逻辑

| 异常场景 | 处理机制 | 用户感知 |
|----------|----------|----------|
| 列表加载失败 | console.error(DEV) + 请求拦截器显示错误 | 全局错误提示 |
| 删除用户失败 | catch → console.error(DEV) | 请求拦截器显示错误 |
| 状态切换失败 | catch → message.error + fetchData()回滚 | "状态变更失败，请重试" |
| 导出数据为空 | 正常生成空CSV文件 | 空文件下载 |
| CSV字段含特殊字符 | escapeCsvField() 转义 | 正常导出 |

## 七、与其他页面/组件的交互关联

| 关联组件/页面 | 交互方式 | 数据流向 |
|--------------|----------|----------|
| `DeptTree` | @select 事件 | 选中部门ID → selectedDeptId |
| `UserFormModal` | v-model:visible + @success | record数据 → 弹窗 → success → 刷新 |
| `ResetPasswordModal` | v-model:visible + @success | userId → 弹窗 → success → 刷新 |
| `TableSetting` | usePageTable composable | 列配置/密度/全屏状态共享 |
| `DictSelect` | v-model:value | 字典编码 → 下拉选项 |
| `useDict` | composable | 字典数据缓存 |
| 后端API | getUserList/deleteUser/changeUserStatus/exportUsers | CRUD操作 |

## 八、用户体验细节

| 细节 | 说明 |
|------|------|
| 静默刷新 | 筛选操作不显示loading动画，避免视觉闪烁 |
| 筛选变更提示 | hasPendingChanges 脉冲动画提醒用户点击查询生效 |
| 删除确认 | Popconfirm 二次确认，防止误操作 |
| 导出BOM头 | CSV添加\ufeff，解决Excel中文乱码 |
| 文件名日期化 | users_2026-05-12.csv 格式 |
| Blob内存释放 | URL.revokeObjectURL() 防止内存泄漏 |
| ID类型统一 | 所有id转Number，避免字符串比较问题 |
| 分页回退 | 删除最后一页最后一条时自动回退一页 |
