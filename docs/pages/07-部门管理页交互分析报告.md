## 一、页面概述

### 1.1 页面定位

部门管理页以树形表格展示组织架构层级，支持节点的增删改查、全局展开/折叠、按名称搜索高亮以及状态切换。页面默认展开所有节点便于浏览。

### 1.2 页面功能模块划分

| 模块 | 位置 | 功能 |
|------|------|------|
| 表格工具栏 | 表格上方 | 新增、展开/折叠按钮 + 列设置 |
| 树形表格 | 页面主体 | 树形展示部门数据（禁用分页，横向滚动1300px） |
| 搜索筛选 | 名称列 | 自定义筛选下拉框（搜索+高亮） |
| 部门表单弹窗 | 模态层 | 新增/编辑部门（含父级ID传递） |
| 部门用户弹窗 | 模态层 | 查看部门下属成员列表 |

### 1.3 依赖组件

| 组件 | 用途 |
|------|------|
| `DeptFormModal` | 部门新增/编辑表单弹窗 |
| `DeptUsersModal` | 部门成员查看弹窗 |
| `TableSetting` | 表格列设置 |
| `usePageTable` | 表格设置管理 |
| `useTreeSearch` | 树形搜索+高亮 |

## 二、用户操作流程

### 2.1 页面初始化

```
onMounted → fetchData()
    ├── loading = true
    ├── getDeptList() → 数据映射（id转Number）
    ├── treeData = tableData (供弹窗使用)
    ├── expandedRowKeys = collectExpandableIds(所有可展开节点) → 默认全部展开
    └── loading = false
```

### 2.2 新增部门/子部门

```
新增顶级 → currentRecord=null, parentId=undefined → 打开弹窗
新增子级 → currentRecord=null, parentId=record.id → 打开弹窗（父级ID传递）
```

### 2.3 编辑/删除/状态切换

| 操作 | 触发 | 确认方式 | 成功后 |
|------|------|----------|--------|
| 编辑 | 点击"修改" | 无 | 打开弹窗(预填数据) |
| 删除 | 点击"删除" | Popconfirm确认 | fetchData() |
| 状态切换 | Switch点击 | 直接执行 | message + fetchData() |
| 查看成员 | 点击"查看成员" | 无 | 打开DeptUsersModal |

### 2.4 展开/折叠控制

- **展开**：collectExpandableIds() 递归收集 → expandedRowKeys赋值
- **折叠**：expandedRowKeys = []
- **单个节点**：handleExpand() 在数组中add/remove

### 2.5 搜索高亮

名称列的筛选下拉框支持搜索，防抖300ms，匹配后自动展开祖先节点并高亮显示匹配文本。

### 3.1 表格列

| 列名 | 特性 |
|------|------|
| 名称 | 搜索高亮 + 自定义筛选下拉 |
| 部门编码 | 纯展示 |
| 负责人/电话/邮箱 | 纯展示 |
| 排序 | 居中数字 |
| 状态 | Switch开关 |
| 创建时间 | 纯展示 |
| 操作 | 添加/修改/查看成员/删除（删除红字danger-link） |

### 3.2 展开图标动画

右箭头（CaretRightOutlined），展开时旋转90度，transition: transform 0.2s ease-in-out。
