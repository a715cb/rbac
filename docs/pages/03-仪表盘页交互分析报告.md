# RBAC 权限管理系统 - 仪表盘页交互分析报告

**文档版本**：1.0 | **生成日期**：2026-05-12 | **页面路径**：`src/pages/dashboard/index.vue`

## 一、页面概述

### 1.1 页面定位

仪表盘页是用户登录成功后的默认落地页面，提供系统核心数据的概览统计和功能介绍。页面采用简洁的卡片式布局，让用户快速了解系统当前状态。

### 1.2 页面功能模块划分

| 模块 | 位置 | 功能 |
|------|------|------|
| 统计卡片区 | 页面顶部 | 4列响应式统计卡片（用户/角色/菜单/部门总数） |
| 欢迎内容区 | 统计卡片下方 | 系统简介 + 功能列表说明 |

### 1.3 统计卡片详细

| 卡片 | 数据字段 | 前端图标 | 颜色主题 |
|------|----------|----------|----------|
| 用户总数 | user_total | UserOutlined | 绿色 #3f8600 |
| 角色总数 | role_total | TeamOutlined | 红色 #cf1322 |
| 菜单总数 | menu_total | MenuOutlined | 蓝色 #4073fa |
| 部门总数 | dept_total | WifiOutlined | 橙色 #faad14 |

## 二、用户操作流程

### 2.1 页面加载流程

```
路由导航到 /dashboard
    │
    ▼
组件挂载 onMounted()
    │
    ▼
fetchStats() 执行
    │
    ├── loading = true → 用户总数卡片显示 Spin 加载动画
    │
    ├── 调用 getDashboardStats() API
    │   ├── 成功 → 更新 stats 响应式对象 → 4张卡片数值刷新
    │   └── 失败 → message.warning('统计数据加载失败')
    │
    └── finally → loading = false → Spin 动画消失
```

### 2.2 用户可执行操作

| 操作 | 说明 |
|------|------|
| 查看统计数据 | 页面加载自动获取，无需用户操作 |
| 阅读功能介绍 | 静态内容，无交互 |
| 通过侧边栏导航到其他页面 | 由 DefaultLayout 的 Sidebar 处理 |

## 三、交互触发条件与状态反馈

### 3.1 数据加载状态

| 状态 | 视觉反馈 | 触发条件 |
|------|----------|----------|
| 加载中 | 用户总数卡片后缀显示 `<a-spin size="small" />` | loading=true |
| 加载成功 | 4张卡片更新为实际数值 (precision:0 显示整数) | API 返回数据 |
| 加载失败 | 顶部显示 warning 消息提示 | API 请求失败 |

### 3.2 数据映射

| 后端字段 | 前端响应式变量 | 显示格式 |
|----------|---------------|----------|
| res.data.user_total | stats.userTotal | 整数，绿色 |
| res.data.role_total | stats.roleTotal | 整数，红色 |
| res.data.menu_total | stats.menuTotal | 整数，蓝色 |
| res.data.dept_total | stats.deptTotal | 整数，橙色 |

### 3.3 无交互操作

仪表盘页本身不包含用户可触发的 CRUD 操作，所有数据操作通过侧边栏导航到其他页面完成。

## 六、异常处理逻辑

| 异常场景 | 处理机制 | 用户感知 |
|----------|----------|----------|
| API请求失败 | catch → message.warning | "统计数据加载失败"警告提示 |
| 响应数据为空 | res.data 判空 → 保持初始值 0 | 卡片显示 0 |
| 网络超时 | 请求拦截器统一处理 | 全局错误提示 |

## 七、与其他页面/组件的交互关联

| 关联组件/页面 | 交互方式 | 说明 |
|--------------|----------|------|
| DefaultLayout | 作为子路由渲染 | 被 router-view 渲染在内容区 |
| Sidebar 菜单 | 路由导航 | 用户通过侧边栏离开仪表盘 |
| getDashboardStats API | 数据获取 | 向后端请求统计数据 |
