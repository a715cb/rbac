## 一、页面概述

### 1.1 页面定位

登录日志页是审计监控模块，展示所有用户的登录记录。支持多维度筛选（关键词+状态+时间范围）、批量删除、全部清除和CSV导出功能。操作按钮受权限指令 `v-auth` 控制。

### 1.2 页面功能模块划分

| 模块 | 位置 | 功能 |
|------|------|------|
| 搜索表单区 | 页面顶部 | 关键词（用户名/IP）+ 状态 + 时间范围筛选 |
| 工具栏 | 表格上方 | 清除日志、批量删除（条件显示）、导出 + 列设置 |
| 数据表格 | 页面主体 | 分页表格 + 行选择(row-selection) |

### 1.3 依赖组件

| 组件 | 用途 |
|------|------|
| `TableSetting` | 表格列设置 |
| `useTableSetting` + `createTableSettingContext` | 表格状态管理 |

## 二、用户操作流程

### 2.1 页面初始化

```
onMounted → fetchData()
    ├── 构建 LoginLogQuery 参数（含日期格式化 YYYY-MM-DD）
    ├── loading = true
    ├── getLoginLogList(params)
    │   ├── 成功 → tableData + pagination.total 更新
    │   └── 失败 → console.error(DEV)
    └── loading = false
```

### 2.2 搜索筛选

| 筛选维度 | 组件 | 特性 |
|----------|------|------|
| 关键词字段选择 | a-select | username / ip 二选一 |
| 关键词输入 | a-input (input-group) | placeholder动态变化 |
| 状态 | a-select | 成功(1)/失败(0) |
| 登录时间 | a-range-picker | 预设：今天/本周/本月/上月 |

### 2.3 行选择与批量操作

```
用户勾选表格行 → selectedRowKeys 更新
    │
    ├── 选中行数 > 0 → "批量删除"按钮显示
    │   └── 点击 → Modal.confirm确认 → deleteLoginLog(ids)
    │       └── 成功 → message显示删除条数 + 清空选择 + fetchData()
    │
    └── 未选中 → 导出按钮导出当前页数据
```

### 2.4 全部清除

```
"清除日志"按钮 (v-auth="admin:login-log:clear")
    └── Modal.confirm确认（danger类型）→ clearLoginLog()
        └── 成功 → selectedRowKeys清空 + pagination回退 + fetchData()
```

### 2.5 CSV导出

支持两种模式：
- **未选择**：导出当前表格所有数据
- **已选择**：仅导出选中的数据

添加BOM头 `\ufeff` 解决中文乱码，文件名 `login_logs_YYYY-MM-DD.csv`。

## 三、交互触发条件与状态反馈

### 3.1 操作系统图标

| 系统 | 图标 | 颜色 |
|------|------|------|
| macOS (含"mac") | AppleFilled | #aaa (灰色) |
| Windows (其他) | WindowsFilled | #40a9ff (蓝色) |

```typescript
const isMac = (os) => os?.toLowerCase().includes('mac')
```

### 3.2 登录状态标签

| 状态值 | 标签颜色 | 文本 |
|--------|----------|------|
| 1 | green | 成功 |
| 0 | red | 失败 |

### 3.3 日期预设

| 预设 | 值 |
|------|-----|
| 今天 | [dayjs(), dayjs()] |
| 本周 | [周一开始, 周日结束] |
| 本月 | [月初, 月末] |
| 上月 | [上月月初, 上月月末] |

### 3.4 权限控制

| 按钮 | 权限标识 | 显示条件 |
|------|----------|----------|
| 清除日志 | `admin:login-log:clear` | v-auth指令 |
| 批量删除 | `admin:login-log:delete` | v-auth + selectedRowKeys.length>0 |

### 3.5 关键词placeholder动态变化

```typescript
const keywordPlaceholder = computed(() => {
    return searchForm.keywordField === 'username'
        ? '请输入用户名搜索' : '请输入IP地址搜索'
})
```
