## 一、页面概述

### 1.1 页面定位

操作日志页属于审计监控模块，记录所有用户的操作行为。支持多维度筛选（操作人+请求方式+IP+时间）、行选择批量操作、全部清除和CSV导出。其中"请求参数"列通过Popover悬浮展示JSON格式的请求数据，并使用highlightjs进行代码高亮。

### 1.2 页面功能模块划分

| 模块 | 位置 | 功能 |
|------|------|------|
| 搜索表单区 | 页面顶部 | 操作人/请求方式/IP/时间范围四维筛选 |
| 工具栏 | 表格上方 | 清除日志、批量删除（条件显示）、导出 + 列设置 |
| 数据表格 | 页面主体 | 分页表格 + 行选择 |
| 参数预览 | 表格内Popover | JSON参数悬浮查看（highlightjs高亮） |

### 1.3 依赖组件

| 组件 | 用途 |
|------|------|
| `TableSetting` | 表格列设置 |
| `useTableSetting` + `createTableSettingContext` | 表格状态管理 |
| `Popover` | 请求参数悬浮展示 |
| `highlightjs` | JSON代码高亮 |

## 二、用户操作流程

### 2.1 页面初始化

```
onMounted → Promise.all([fetchUserOptions(), fetchData()])
    ├── fetchUserOptions → getUserList({ page:1, limit:1000 })
    │   └── 填充 userOptions 下拉数据
    └── fetchData → getOperateLogList(params)
        └── 成功 → tableData + pagination.total 更新
```

### 2.2 搜索筛选

| 维度 | 组件 | 数据来源 |
|------|------|----------|
| 操作人 | a-select | userOptions（用户列表API） |
| 请求方式 | a-select | GET/POST/PUT/DELETE静态选项 |
| IP地址 | a-input | 用户输入 |
| 操作时间 | a-range-picker | 预设：今天/本周/本月/上月 |

### 2.3 请求参数悬浮查看

```
用户hover/click"查看"标签
    │
    ▼
Popover弹出 (placement="top")
    ├── content: <highlightjs autodetect code={record.param || '{}'} />
    ├── 最大宽度400px × 最大高度300px
    └── 代码区域可滚动
```

### 2.4 批量操作与导出

与登录日志页相同的模式：行选择 → 批量删除 / 选中/全部导出。

## 三、交互触发条件与状态反馈

### 3.1 请求方式标签颜色

| 方法 | Tag颜色 |
|------|---------|
| GET | green |
| POST | blue |
| PUT | orange |
| DELETE | red |
| 其他 | default |

### 3.2 操作状态标签

| 状态 | 颜色 | 文字 |
|------|------|------|
| 1 | green | 成功 |
| 0 | red | 失败 |

### 3.3 参数查看交互

- **触发方式**：点击蓝色"查看"Tag
- **展示位置**：Popover 顶部对齐
- **内容渲染**：highlightjs自动检测语言
- **样式适配**：暗黑模式下背景 #f0f0f0
- **滚动**：内容超出时 overflow: auto

```html
<Popover placement="top">
    <template #content>
        <highlightjs autodetect code={record.param || '{}'} class="code" />
    </template>
    <Tag color="blue" class="text-[12px]">查看</Tag>
</Popover>
```

### 3.4 权限控制

| 按钮 | 权限 | 显示条件 |
|------|------|----------|
| 清除日志 | `admin:operation-log:clear` | v-auth |
| 批量删除 | `admin:operation-log:delete` | v-auth + selectedRowKeys.length>0 |

## 六、异常处理

| 异常 | 处理 |
|------|------|
| 用户列表加载失败 | userOptions=[]，下拉为空 |
| 日志列表加载失败 | console.error(DEV) |
| 清除日志失败 | message.error |
| 批量删除失败 | message.error |
| 请求参数为空 | 默认显示 '{}' |
