# RBAC 权限管理系统 - 错误页面交互分析报告

**文档版本**：1.0 | **生成日期**：2026-05-12 | **页面路径**：`src/pages/error/403.vue` + `src/pages/error/404.vue`

---

## 一、页面概述

### 1.1 页面定位

错误页面是系统的容错兜底机制，包含403（权限不足）和404（页面不存在）两个页面。它们不依赖DefaultLayout，独立渲染在全屏居中位置。

### 1.2 两个错误页面对比

| 特性 | 403 页面 | 404 页面 |
|------|----------|----------|
| 错误码 | 403 | 404 |
| 图标 | StopOutlined | QuestionCircleOutlined |
| 主色调 | 红色 (#ff4d4f) | 蓝色 (#4073fa) |
| 提示文字 | 抱歉，您没有权限访问该页面 | 抱歉，您访问的页面不存在 |
| 按钮1 | 返回上一页 | 返回首页 |
| 按钮2 | 返回首页 | - |

---

## 二、403页面交互分析

### 2.1 渲染条件

- 路由守卫检测到用户无权限时跳转
- 由 Vue Router 导航到此路由

### 2.2 用户操作

| 操作 | 触发 | 行为 |
|------|------|------|
| 点击"返回上一页" | goBack() | router.go(-1) 浏览器后退 |
| 点击"返回首页" | goHome() | router.push('/dashboard') |

### 2.3 状态反馈

- 80px红色StopOutlined图标
- 72px粗体红色"403"数字
- 18px灰色提示文字
- 主按钮（蓝色）+ 次按钮（默认）双操作

---

## 三、404页面交互分析

### 3.1 渲染条件

- 用户访问未定义的路由时由Vue Router的catch-all路由匹配
- 由 `path: '/:pathMatch(.*)*'` 捕获

### 3.2 用户操作

| 操作 | 触发 | 行为 |
|------|------|------|
| 点击"返回首页" | goHome() | router.push('/dashboard') |

### 3.3 与403的区别

- 仅有一个操作按钮（返回首页）
- 无"返回上一页"选项（因为404通常来自外部链接或直接URL输入）
- 主题色为蓝色而非红色（404是"找不到"而非"被拒绝"）

## 四、动画效果

两个错误页面均为静态页面，无特殊动画。

| 元素 | 说明 |
|------|------|
| 页面布局 | flex居中，无入场动画 |
| 图标/文字 | 静态渲染 |
| 按钮 | Ant Design默认hover/active效果 |

## 五、响应式适配

```
.error-page {
    height: 100vh;
    display: flex;
    align-items: center;
    justify-content: center;
    background: #f0f2f5;
}
```

- 始终垂直水平居中
- 全视口高度
- 浅灰背景（#f0f2f5）
- 无断点级别的特殊适配，flex居中自动适应各种屏幕

## 六、异常处理

| 异常 | 处理 |
|------|------|
| router.go(-1) 失败 | 浏览器默认行为（无历史则无反应） |
| router.push('/dashboard') 失败 | Vue Router 内部错误处理 |
| dashboard路由不存在 | 理论上不会发生（基础路由） |

---

## 七、与其他页面/组件的交互关联

| 关联 | 方式 |
|------|------|
| Vue Router | 路由匹配/守卫 → 错误页 |
| Dashboard | router.push → 返回正常页面 |
| 上一页（403） | router.go(-1) → 浏览器历史 |

---

## 八、设计意图分析

### 8.1 颜色语义

| 页面 | 颜色 | 语义 |
|------|------|------|
| 403 | 红色 | 警告/拒绝/危险 - 用户被明确拒绝访问 |
| 404 | 蓝色 | 信息/中性 - 资源不存在，非用户过错 |

### 8.2 操作引导

- 403提供"返回上一页" + "返回首页"双路径，覆盖不同用户意图
- 404只提供"返回首页"，简化选择（外部链接来的用户没有"上一页"）
