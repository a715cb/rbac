## 跨页面通用机制

以下交互模式在多个页面中复用，详细实现请参阅各子页面文档。

| 机制 | 使用页面 | 核心实现 |
|------|---------|---------|
| 静默刷新 | 用户管理、角色管理、部门管理等 | `fetchData(true)` 控制 loading 显隐，避免频繁动画闪烁 |
| 字典缓存 | 用户管理、角色管理、API管理等 | `useDict()` 全局 Map 缓存 + 订阅者模式避免重复请求 |
| 表格设置 | 所有列表页 | `usePageTable` 组合式函数管理列显隐、密度、全屏状态 |
| 树形搜索 | 菜单管理、部门管理 | `useTreeSearch` 防抖300ms + 文本高亮 + 自动展开匹配节点 |
| 表单弹窗 | 用户、角色、菜单、部门、字典、API | `watch(visible)` 加载数据 → `nextTick` 重置/填充 → `validate()` 提交 → `destroy-on-close` |
| Token刷新 | 全局（Axios拦截器） | 订阅者模式避免并发刷新，失败自动跳转登录页 |
| 权限指令 | 日志管理等 | `v-auth` / `v-auth-disabled` 控制按钮显隐与禁用 |
